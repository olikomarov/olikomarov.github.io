mutual_friends_inside_BSPb_group.csv:

колонки id1, id2.
Это люди, подписанные на паблик банка "Санкт-Петербург", которые есть друг у друга в друзьях.
каждый ряд - ребро неориентированного графа.
т.е. ряд 236,46309 означает, что у пользователя с айдишником 236 есть в друзьях 46309 
и у пользователя 46309 есть в друзьях id 236. ребра 46309,236 в данных нет. подразумевается, что достаточно одного.



profiles_info.json:
